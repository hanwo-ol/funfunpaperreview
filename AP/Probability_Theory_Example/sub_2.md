### 개념 1: 가산 무한(Countably Infinite)과 Cardinality ℵ₀

강의 자료의 핵심은 다음과 같습니다.

> 어떤 집합 A의 Cardinality가 **ℵ₀ (알레프-널)**이라는 것은, 그 집합의 모든 원소를 자연수 1, 2, 3, ... 에 빠짐없이 그리고 중복 없이 하나씩 **번호를 붙여 나열할 수 있다**는 의미입니다. 이를 수학적으로는 "자연수 집합 N과 집합 A 사이에 **전단사 함수(bijection)**가 존재한다"고 표현하며, 이러한 집합을 **"셀 수 있다 (countable)"**고 합니다.

즉, 문제에 주어진 집합들의 Cardinality가 ℵ₀임을 보이려면, 그 집합의 원소들을 `a₁, a₂, a₃, ...` 와 같은 형태로 나열할 수 있음을 보이면 됩니다.

---

### 문제 풀이

#### **1. {0} ∪ N 의 카디널리티는?** (음이 아닌 정수 집합)

*   **집합:** A = {0, 1, 2, 3, 4, ...}
*   **답:** Cardinality는 **ℵ₀** 입니다.

*   **풀이 (전개):**
    자연수 집합 N = {1, 2, 3, ...}에서 집합 A로 가는 함수 `f: N → A`를 다음과 같이 정의할 수 있습니다.
    **`f(n) = n - 1`**

    이 함수를 통해 A의 모든 원소를 다음과 같이 나열할 수 있습니다.
    *   `f(1)` = 0  (첫 번째 원소 `a₁`)
    *   `f(2)` = 1  (두 번째 원소 `a₂`)
    *   `f(3)` = 2  (세 번째 원소 `a₃`)
    *   ...
    *   `f(n)` = n - 1 (`n`번째 원소 `aₙ`)

    이 함수는 N의 모든 원소를 A의 모든 원소에 빠짐없이 그리고 중복 없이 일대일로 대응시키는 **전단사 함수**입니다. 따라서 개념 1에 의해 {0} ∪ N의 Cardinality는 N의 Cardinality와 같은 **ℵ₀** 입니다.
    *(이는 무한집합에 유한 개의 원소를 추가해도 Cardinality는 변하지 않음을 보여주는 예시입니다.)*

#### **2. 짝수집합의 카디널리티는?**

*   **집합:** A = {2, 4, 6, 8, ...}
*   **답:** Cardinality는 **ℵ₀** 입니다.

*   **풀이 (전개):**
    자연수 집합 N에서 짝수 집합 A로 가는 함수 `f: N → A`를 다음과 같이 정의할 수 있습니다.
    **`f(n) = 2n`**

    이 함수를 통해 A의 모든 원소를 나열하면 다음과 같습니다.
    *   `f(1)` = 2  (`a₁`)
    *   `f(2)` = 4  (`a₂`)
    *   `f(3)` = 6  (`a₃`)
    *   ...

    이 함수는 명백히 전단사 함수입니다. 따라서 짝수 집합은 "셀 수 있으며" Cardinality는 **ℵ₀** 입니다.
    *(이는 무한집합의 경우, 자기 자신의 진부분집합(proper subset)과도 Cardinality가 같을 수 있다는 중요한 성질을 보여줍니다.)*

#### **3. 홀수집합의 카디널리티는?**

*   **집합:** A = {1, 3, 5, 7, ...}
*   **답:** Cardinality는 **ℵ₀** 입니다.

*   **풀이 (전개):**
    자연수 집합 N에서 홀수 집합 A로 가는 함수 `f: N → A`를 다음과 같이 정의할 수 있습니다.
    **`f(n) = 2n - 1`**

    이 함수를 통해 A의 모든 원소를 나열하면 다음과 같습니다.
    *   `f(1)` = 1  (`a₁`)
    *   `f(2)` = 3  (`a₂`)
    *   `f(3)` = 5  (`a₃`)
    *   ...

    이 또한 전단사 함수이므로, 홀수 집합의 Cardinality는 **ℵ₀** 입니다.

#### **4. 정수집합의 카디널리티는?**

*   **집합:** Z = {..., -2, -1, 0, 1, 2, ...}
*   **답:** Cardinality는 **ℵ₀** 입니다.

*   **풀이 (전개):**
    정수 집합은 양쪽으로 무한히 뻗어있어 한 번에 나열하기 어려워 보이지만, 다음과 같은 규칙으로 모든 원소를 셀 수 있습니다.
    **0부터 시작하여 양수와 음수를 번갈아가며 나열합니다.**

    `0, 1, -1, 2, -2, 3, -3, ...`

    이를 순서대로 `a₁, a₂, a₃, ...` 라고 번호를 붙이면:
    *   `a₁` = 0
    *   `a₂` = 1
    *   `a₃` = -1
    *   `a₄` = 2
    *   `a₅` = -2
    *   ...

    이러한 방식으로 모든 정수를 빠짐없이 나열할 수 있으므로, 정수 집합은 "셀 수 있습니다". 따라서 정수 집합의 Cardinality는 **ℵ₀** 입니다.

#### **5. {3n - 2 : n = 1, 2, 3, ...}의 카디널리티는?**

*   **집합:** A = {1, 4, 7, 10, ...} (첫째항이 1이고 공차가 3인 등차수열)
*   **답:** Cardinality는 **ℵ₀** 입니다.

*   **풀이 (전개):**
    이 문제는 집합의 정의 자체가 이미 자연수 n을 이용한 함수로 주어져 있습니다.
    **`f(n) = 3n - 2`**

    이 함수 `f`는 자연수 집합 N에서 집합 A로 가는 함수입니다.
    *   `f(1)` = 1
    *   `f(2)` = 4
    *   `f(3)` = 7
    *   ...
    함수의 정의 자체가 `n` 값 하나에 `3n-2` 값 하나가 대응되도록 만들어졌으므로, 이 함수는 자연스럽게 **전단사 함수**가 됩니다. 따라서 이 집합의 Cardinality는 **ℵ₀** 입니다.

### 심화 학습: Durrett 교재와의 연결

이 개념은 Durrett 교재 **Chapter 1**의 **이산 확률 공간 (Discrete probability spaces, Example 1.1.2)**을 이해하는 데 결정적인 역할을 합니다.

*   **교재의 핵심:** Ω(표본 공간)가 **가산 집합(countable set)**, 즉 Cardinality가 유한하거나 **ℵ₀**일 때, 우리는 각 원소 ω에 확률 `p(ω)`를 부여하고, 어떤 사건 A의 확률을 `P(A) = Σ_{ω∈A} p(ω)` (사건에 속한 원소들의 확률 합)으로 정의할 수 있습니다.

*   **연결점:**
    *   오늘 푼 모든 문제의 집합들(정수, 짝수, 홀수 등)은 모두 Cardinality가 ℵ₀ 이므로 **가산 집합**입니다.
    *   따라서 만약 이 집합들을 확률 실험의 표본 공간(Ω)으로 사용한다면, 우리는 각 정수, 각 짝수, 각 홀수 하나하나에 확률을 부여하는 방식으로 확률 모델을 만들 수 있습니다.
    *   예를 들어, 표본 공간이 정수 집합(Z)인 경우, 각 정수 `k`가 나올 확률 `p(k)`를 정의하고, "양수가 나올 확률"은 `Σ_{k=1 to ∞} p(k)` 와 같이 계산할 수 있습니다.
    *   이것이 바로 **이산 확률 변수(Discrete Random Variable)**의 이론적 배경이 됩니다.

이 방법은 '건너뛰는' 과정을 없애고, 처음부터 중복이 없는 집합을 정의한 뒤 그 집합을 세는 방식입니다.

**핵심 아이디어:** 모든 양의 유리수는 **기약분수** 형태로 유일하게 표현될 수 있습니다. (예: 0.5는 1/2, 2/4, 3/6 등으로 표현되지만, 기약분수 표현은 1/2 하나뿐입니다.)

1.  **고유 표현 집합 S 정의:**
    양의 유리수와 일대일로 대응되는, 서로소인 자연수 쌍의 집합 S를 다음과 같이 정의합니다.
    S = { (p, q) | p, q ∈ N 이고, gcd(p, q) = 1 }
    여기서 `N`은 자연수 집합, `gcd(p, q)`는 p와 q의 최대공약수입니다. `gcd(p, q) = 1`이라는 조건이 바로 `p/q`가 기약분수임을 의미합니다.

2.  **증명 목표 변경:**
    이제 우리의 목표는 `|Q⁺| = |N|`임을 보이는 대신, **`|S| = |N|`** 임을 보이는 것으로 바뀝니다. 왜냐하면 집합 S와 양의 유리수 집합 Q⁺ 사이에는 `f(p, q) = p/q` 라는 명백한 **전단사 함수(bijection)**가 존재하기 때문입니다.

3.  **S를 세는 방법 (식으로 표현):**
    집합 S는 자연수의 쌍으로 이루어진 집합 `N × N`의 부분집합입니다. (`S ⊂ N × N`)
    우리는 이미 `N × N`이 가산 집합임을 알고 있습니다 (원래의 대각선 논법).

    집합론의 기본 정리에 따르면, **"가산 무한 집합의 무한 부분집합은 가산 무한이다."**

    *   S는 `(1, 1), (1, 2), (2, 1), (1, 3), (3, 1), ...` 등을 포함하는 무한 집합입니다.
    *   S는 가산 무한 집합 `N × N`의 부분집합입니다.
    *   따라서 S는 가산 무한 집합입니다. 즉, `|S| = ℵ₀` 입니다.

    이것만으로도 증명은 충분하지만, 더 구체적인 함수로 표현하자면 다음과 같이 할 수 있습니다.

    *   기존의 대각선 논법에서 사용한, `N × N`의 모든 원소를 나열하는 경로를 그대로 따릅니다.
    *   이 경로를 따라가면서, `gcd(p, q) = 1`인 쌍 `(p, q)`를 만날 때마다 순서대로 자연수 번호를 붙입니다.

    예를 들어:
    *   **(1, 1):** gcd(1, 1) = 1. **첫 번째**로 센다. (→ 1/1)
    *   **(1, 2):** gcd(1, 2) = 1. **두 번째**로 센다. (→ 1/2)
    *   **(2, 1):** gcd(2, 1) = 1. **세 번째**로 센다. (→ 2/1)
    *   **(3, 1):** gcd(3, 1) = 1. **네 번째**로 센다. (→ 3/1)
    *   **(2, 2):** gcd(2, 2) = 2 ≠ 1. **건너뛴다.**
    *   **(1, 3):** gcd(1, 3) = 1. **다섯 번째**로 센다. (→ 1/3)
    *   ...

    이 과정은 모든 기약분수를 정확히 한 번씩만 세게 되므로, **자연수 N과 집합 S (즉, Q⁺) 사이에 전단사 함수가 존재함**을 구성적으로 보여줍니다.
