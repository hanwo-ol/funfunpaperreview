문제에서 주어진 확률변수 $X$의 분포를 이용해 기댓값과 분산을 먼저 계산해야 합니다.

1. 기댓값 $E(X)$ 계산주어진 확률질량함수는 다음과 같습니다.
$P(X=1) = 1/18$
$P(X=2) = 16/18$
$P(X=3) = 1/18$
기댓값 $E(X)$는 다음과 같이 계산됩니다.

$$E(X) = 1 \cdot \frac{1}{18} + 2 \cdot \frac{16}{18} + 3 \cdot \frac{1}{18} = \frac{1 + 32 + 3}{18} = \frac{36}{18} = 2$$

2. 분산 $Var(X)$ 계산분산을 구하기 위해 $E(X^2)$를 먼저 계산합니다.

$$E(X^2) = 1^2 \cdot \frac{1}{18} + 2^2 \cdot \frac{16}{18} + 3^2 \cdot \frac{1}{18} = \frac{1 + 64 + 9}{18} = \frac{74}{18} = \frac{37}{9}$$

따라서 분산은 다음과 같습니다.

$$Var(X) = E(X^2) - \{E(X)\}^2 = \frac{37}{9} - 2^2 = \frac{37}{9} - \frac{36}{9} = \frac{1}{9}$$

3. 주어진 방정식 설정 및 확률 분석문제에서 만족해야 하는 방정식은 다음과 같습니다.

$$P(|X - 2| \ge \delta) = \frac{1/9}{\delta^2}$$

(단, 우변의 분모에 $\delta$가 있으므로 $\delta \neq 0$ 이어야 합니다.)좌변의 확률을 구하기 위해 $|X - 2|$가 가질 수 있는 값과 그 확률을 확인해 봅니다.
$X=1$일 때, $|1-2| = 1$ (확률 $1/18$)$X=2$일 때, $|2-2| = 0$ (확률 $16/18$)$X=3$일 때, $|3-2| = 1$ (확률 $1/18$)즉, $|X - 2|$는 오직 0 또는 1의 값만 가질 수 있으며, 각각의 확률은 다음과 같습니다.
$P(|X - 2| = 0) = \frac{16}{18} = \frac{8}{9}$
$P(|X - 2| = 1) = \frac{1}{18} + \frac{1}{18} = \frac{2}{18} = \frac{1}{9}$

4. $\delta$의 범위에 따른 케이스 분류 및 풀이$\delta$가 양수라는 보장이 없으므로, $\delta$의 범위를 나누어 방정식을 풉니다.

Case 1: $\delta < 0$ 인 경우절댓값 $|X - 2|$는 항상 0 이상이므로, 음수인 $\delta$에 대해서 $|X - 2| \ge \delta$는 **항상 참(True)** 입니다.
따라서 좌변의 확률은 1이 됩니다.

$$1 = \frac{1}{9\delta^2}$$

$$9\delta^2 = 1 \implies \delta^2 = \frac{1}{9}$$

$$\delta = \frac{1}{3} \text{ 또는 } \delta = -\frac{1}{3}$$

현재 $\delta < 0$인 경우를 가정하고 있으므로, 이 구간에서의 해는 **$\delta = -1/3$** 입니다.

Case 2: $0 < \delta \le 1$ 인 경우이 구간에서는 $|X - 2|$가 가질 수 있는 값(0, 1) 중, 오직 1만이 $\delta$보다 크거나 같습니다.따라서 좌변의 확률은 $P(|X - 2| = 1)$과 동일합니다.

$$\frac{1}{9} = \frac{1}{9\delta^2}$$

$$\delta^2 = 1$$

$$\delta = 1 \text{ 또는 } \delta = -1$$

현재 $0 < \delta \le 1$인 경우를 가정하고 있으므로, 이 구간에서의 해는 **$\delta = 1$** 입니다.

Case 3: $\delta > 1$ 인 경우이 구간에서는 $|X - 2|$가 가질 수 있는 값(0, 1) 모두 $\delta$보다 작습니다.따라서 $|X - 2| \ge \delta$를 만족할 확률은 0입니다.

$$0 = \frac{1}{9\delta^2}$$

이를 만족하는 실수 $\delta$는 존재하지 않습니다.

결론)양수 가정을 배제하고 모든 실수 범위에서 주어진 등식을 만족하는 $\delta$를 찾은 결과, 정답은 다음과 같습니다.$\delta = 1$ 또는 $\delta = -\frac{1}{3}$
